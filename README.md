# 🌴 The Wild Oasis

A modern hotel management web application designed to streamline cabin bookings, guest management, and business insights — built with a clean UI and powerful backend tools.

---

## 🚀 Features

* 🏡 **Cabin Management**
  Create, edit, and manage cabins with images, pricing, and availability.

* 📅 **Booking System**
  Track reservations, check-ins, and check-outs with ease.

* 👤 **Guest Management**
  Store and manage guest details efficiently.

* 📊 **Dashboard Analytics**
  Get insights into bookings, occupancy rates, and revenue.

* 🔐 **Authentication & Authorization**
  Secure login system with role-based access.

* 🌙 **Dark Mode Support** *(if applicable)*
